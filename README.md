# Handwritten Arabic Digit Recognition

This project implements a Convolutional Neural Network (CNN) model to recognize handwritten Arabic digits.

## Dataset
This project uses a publicly available dataset:

https://data.mendeley.com/datasets/5hpkf8v7bg/1

The dataset contains images of handwritten Arabic numerals collected from multiple writers. 
It includes digits from 0 to 9 and captures natural variations in handwriting style.

Note: The dataset is not included in this repository due to its large size.

## Model
The model is based on a Convolutional Neural Network (CNN) architecture trained to classify digits from 0 to 9.

Test Accuracy: ~94%
